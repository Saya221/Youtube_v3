# README

* Description
  - An example app using google-api-client/googleauth
  - Ruby version 3.0.0
  - Rails 7

* Starting rails app
  - bundle install
  - rails s

* Code rules
  Running before push:
    - brakeman (for normal injections)
    - rubocop -A (for coding style)
    - rspec (check file coverage at least > 80%)
